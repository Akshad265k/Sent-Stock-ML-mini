# Market Pulse Dashboard

## Project Overview

Market Pulse Dashboard is a modern web application designed to visualize and analyze market trends using an interactive and responsive UI. The project focuses on clean design, fast performance, and scalability.

---

## Tech Stack

This project is built using:

- **Vite** – Fast build tool and dev server  
- **TypeScript** – Type-safe JavaScript  
- **React** – UI library  
- **shadcn/ui** – Accessible and customizable components  
- **Tailwind CSS** – Utility-first styling framework  

---

## Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js**
- **npm** (comes with Node.js)

---

### Run the Project Locally

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <PROJECT_FOLDER_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
